💻 Technology Stack

CrowdMapper is built with modern, enterprise-grade technologies:

Backend Framework

Spring Boot 3.2.0 - Enterprise Java framework for building RESTful APIs and web services with rapid development capabilities.

Architecture Pattern

Spring MVC & Spring Data JPA - Model View Controller pattern for organized structure and simplified database operations.

Database

H2 (Development) / PostgreSQL Ready - In-memory relational database for development with production-ready PostgreSQL support.

Frontend

Thymeleaf, HTML5 & CSS3 - Server-side template engine with modern web standards for dynamic, responsive interfaces.

Client-side Logic

JavaScript - Vanilla JS for interactive features, form validation, and dynamic user experience.

Build & Deployment

Apache Maven & Java 17 - Dependency management with latest LTS version. Spring Boot JAR for containerization.

Design Principles

Microservices Ready - RESTful APIs, stateless design, and responsive UI supporting mobile-first approach.

Development Tools

Spring DevTools - Live reload for faster development iteration and immediate feedback.
