🧮 How Priority is Calculated

Priority Score Formula:

Priority Score = (Confirmations − Rejections) + Severity Score

Where:

  • Confirmations: Number of citizens who verified the problem exists
  • Rejections: Number who voted the problem doesn't exist
  • Severity Score: High=+10, Medium=+5, Low=0

✓ This ensures urgent AND verified problems get maximum attention.

🔒 Transparency & Trust – How We Prevent Misuse

👥 Crowd Verification

Problems must be verified by multiple citizens. Fake reports won't gain votes and stay deprioritized.

🛡️ Admin Moderation

Administrators review high-priority issues before action. Manual oversight prevents spam/abuse.

📊 Public Visibility

All verification counts are visible. Citizens see exactly how many people confirmed each issue.

🔍 Transparent Status Updates

Issues progress through clear statuses: Reported → Verified → In Progress → Resolved.
